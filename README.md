# Project Name

Solar Simulation.

## Getting Started

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed.

### Installation

1. Clone the repository:

git clone https://github.com/EmranHayder/solar-simulation.git
cd solar-simulation

2. Install dependencies:

npm install

3. Start the development server:

npm run dev

4. Localhost Port

http://localhost:5173/
